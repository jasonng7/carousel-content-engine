Use this file as the neutral onboarding guide for any creator, founder, brand, agency, or team using the Carousel Content Engine.

Before generating a carousel, replace the template values below or ask the onboarding questions in `project-instructions.txt`. If a field is unknown, ask briefly instead of inventing a highly personal brand direction.

## 4. Tone Of Voice

Choose 3-6 traits:

- Clear
- Warm
- Direct
- Playful
- Premium
- Analytical
- Opinionated
- Encouraging
- Minimal
- Energetic
- Calm
- Technical
- Conversational
- Editorial

### Use
- Short, skimmable sentences
- Specific examples
- One clear idea per slide
- Plain language before jargon
- Concrete verbs and outcomes

### Avoid
- Generic motivational quotes
- Overly corporate phrasing
- Fake urgency
- Dense paragraphs
- Buzzwords without examples
- Claims the brand cannot support

## 8. Default Carousel Brief

When the user has completed the onboarding answers, generate a carousel using this neutral brief:

> Create a vertical social carousel for `[brand_name]`. The carousel should help `[audience]` understand `[topic]` and take `[desired_action]`. Match the brand tone, color palette, typography direction, and content pillars from this profile. Keep the copy concise, the design export-ready, and the final CTA aligned with the user's goal.
